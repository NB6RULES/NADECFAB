    Git Installation on Windows#

    This guide walks you through installing Git on a Windows machine.

    Download Git#

    1. Visit the official Git website: https://git-scm.com
    2. Click the Download for Windows button
    3. The installer will automatically detect your system (32-bit or 64-bit) and download the appropriate version

    Run the Installer#

    1. Open the downloaded .exe file
    2. Click Next to proceed through the installation wizard

    Installation Options#

    Step 1: Select Components#

    • Keep the default selections checked:
      • ✓ Git Bash Here
      • ✓ Git GUI Here
      • ✓ Git LFS (Large File Support)
      • ✓ Associate .git* configuration files with the default text editor

    Step 2: Choose Default Editor#

    • Select your preferred text editor (Notepad++, VS Code, or Vim)
    • If unsure, choose Nano as it’s beginner-friendly

    Step 3: Configure PATH Environment#

    • Select Git from the command line and also from 3rd-party software
    • This allows you to use Git in any terminal application

    Step 4: HTTPS Transport Backend#

    • Select Use the native Windows Secure Channel library
    • This uses Windows’ built-in SSL/TLS for secure connections

    Step 5: Line Ending Conversions#

    • Select Checkout as-is, commit as-is (recommended for cross-platform projects)
    • Or select Checkout Windows-style, commit Unix-style if you’re working on Windows only

    Step 6: Terminal Emulator#

    • Select Use Windows’ default console window
    • MinTTY is also a good option if you prefer a Unix-like terminal

    Step 7: Extra Options#

    • Check Enable file system caching for better performance
    • Check Enable Git Credential Manager for easier authentication

    Complete Installation#

    1. Click Install to start the installation
    2. Wait for the process to complete (usually takes 1-2 minutes)
    3. Click Finish

    Verify Installation#

    Open Command Prompt or PowerShell and run:

    git --version

    You should see output like: git version 2.43.0.windows.1

    Initial Configuration#

    Configure your Git identity for commits:

    git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

    Verify the configuration:

    git config --global --list
